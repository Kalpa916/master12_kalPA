BootStrap
---------
> front-end framewrok (free), and it was developed by "Twitter" company.
> frame work is a coll of ready-made "clssses" (css selectors), which can be used in webpages to apply styles to the elements easily.
   BS => css + js
	          
> BS can be used w/o CSS knowledge also,   to customize the BS styles, CSS knowledge is req.

> BS provides Responsive web design (inbuilt) & mobile-first web sties.

> BS was devloped based on "jQuery" and "Popper".
> jQuery is JavaScript lib, which provides set of functions to perform DOM manipulations.
> Popper is a javaScript library, which provides a set of functions to display popup messages.

				 
				
Bootstrap 
---------
1. It is a free front-end framework for faster and easier web development.

2. It combines HTML, CSS and JS based design templates for typography, forms, buttons, tables, navigation, image carousels and numerous other components, as well as JavaScript plugins.

3. It provides you with a convenient approach for creating responsive designs.

What is Responsive Web Design?
Responsive web design is about creating web sites which automatically adjust themselves to look good on all devices, from small phones to large desktops.



Why use Bootstrap?
-------------------------
1. It is novice-friendly, so you don’t have to master HTML or CSS before diving into it.

2. Its CSS works perfectly on either phones, tablets or desktops thanks to its responsive features.

3. Its core framework styles are preset to mobile-first approach.

4. All the modern browsers are entirely compatible with Bootstrap (Chrome, Firefox, Internet Explorer, Microsoft Edge, Safari, and Opera).

HTML5 doctype
Bootstrap requires the use of the HTML5 doctype. Without it, you’ll see some funky incomplete styling, but including it shouldn’t cause any considerable hiccups.
